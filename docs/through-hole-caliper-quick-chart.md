# Through-Hole Caliper Quick Chart (Inch Screws)

Use this when you measured an existing **through-hole diameter** and want a fast guess for inch screw size.

## How to Use

1. Measure hole ID with calipers (inches preferred).
2. Find that value in the range table below.
3. Start with the suggested screw size.
4. Confirm by checking screw major diameter and fit in real hardware.

## Quick Sanity Rules

- If fit is too tight, move down one fit class (close -> normal -> loose).
- If hole is slotted, worn, plated, or painted, size can read larger than nominal.
- Do not mix inch and metric screws even when diameter seems close.
