# Through-Hole Caliper Quick Chart (Metric Screws)

Use this when you measured an existing **through-hole diameter** and want a fast guess for metric screw size.

## How to Use

1. Measure hole ID with calipers (mm preferred).
2. Find your measured value in the table.
3. Start with the suggested metric screw size.
4. Confirm fit with real hardware and adjust fit class if needed.

## Quick Sanity Rules

- If fit is tighter than expected, compare against close/normal/loose clearance sizes.
- Coatings, paint, burrs, and wear can shift measured hole size.
- Do not substitute inch screws for metric unless explicitly designed for conversion.
