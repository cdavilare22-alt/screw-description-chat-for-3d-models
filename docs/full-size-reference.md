# Full Inch and Metric Size Reference

This page links the complete size datasets and includes quick-glance tables.

## Complete Datasets

- `data/inch_unified_sizes.csv` (UNC/UNF/UNEF coverage from #0 to 1 in)
- `data/metric_iso_sizes.csv` (ISO metric coarse + common fine sizes M1.6 to M30)
- `data/inch_mm_conversion_chart.csv` (fractional inch to mm, 1/64 increments)

## Quick Conversion Formula

- `mm = inches x 25.4`
- `inches = mm / 25.4`
